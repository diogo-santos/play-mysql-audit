@() Play REST API

Play REST API

This is a placeholder page to show you the REST API. Use httpie to post JSON to the application.

To see all events, you can do a GET: 

    http GET localhost:9000/v1/events/

To create new events, do a post 

    http POST localhost:9000/v1/events/ {"userId": 1,"eventTime": [2019,3,20,10,35,55],"userRole": "role","details": "details"}

You can always look at the API directly: /v1/events
